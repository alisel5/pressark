# PressArk bundled fonts

PressArk self-hosts these interface fonts to avoid remote font stylesheet requests in the WordPress admin/front-end panel.

- Inter: SIL Open Font License 1.1, https://github.com/rsms/inter
- Instrument Serif: SIL Open Font License 1.1, https://github.com/Instrument/instrument-serif
- JetBrains Mono: SIL Open Font License 1.1, https://github.com/JetBrains/JetBrainsMono

The WOFF2 files are latin and latin-ext subsets downloaded from the Google Fonts distribution.
See `OFL-1.1.txt` for the bundled font license text.
